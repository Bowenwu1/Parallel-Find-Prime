# Parallel-Find-Prime
利用MPI实现并行的线性筛数法

## 编译
* 串行

    ```shell
    mpic++ normal_find_prime.cpp -o normal_find_prime
    ```

* 并行

    ```shell
    mpic++ parallel_find_prime.cpp -o parallel_find_prime
    ```

## 运行
* 串行

    ```Shell
    mpirun -n 1 normal_find_prime 0 [i]
    ```

* 并行

    ```Shell
    mpirun -n [进程数] --oversubscribe parallel_find_prime [i]
    ```

    * 进程数 ：新建的进程数
    * $i$ : 筛选小于等于 $2^i*10^3$ 的素数
